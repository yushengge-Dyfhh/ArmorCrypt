ArmorCrypt Security Architecture

Layer 1: ChaCha20-Poly1305

High-speed stream cipher with built-in authentication. Provides confidentiality and integrity.

Layer 2: AES-256-GCM

Military-grade symmetric encryption with Galois/Counter Mode for authenticated encryption.

Layer 3: Dual Key Derivation

PBKDF2 (100,000 iterations) + Argon2id (memory-hard) for brute-force resistance.

Layer 4: Temporal Salt System

Time-based salt values ensure identical data encrypts differently at different times.

Layer 5: Asymmetric Envelope

Elliptic Curve Cryptography (P-521) for key exchange and digital signatures.

Layer 6: Integrity Verification

Multi-layer HMAC and digital signatures prevent tampering and ensure data authenticity.

Layer 7: Quantum Resistance

Lattice-based cryptography and large key sizes protect against future quantum attacks.
